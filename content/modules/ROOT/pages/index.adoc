= ChatOps generative AI with LLM, RAG and data science pipelines

Welcome to this demonstration of retrieval-augmented generation (RAG) on Red Hat OpenShift AI. This presenter-led demo shows how organizations build AI-powered chatbots with company-specific knowledge, eliminating expensive model retraining.

== What This Demo Covers

This demonstration focuses on three key areas:

**Business Outcomes:**

* 10Ã— cost reduction compared to LLM retraining
* 70% reduction in customer support escalations
* Minutes (not weeks) to update AI knowledge bases
* Enterprise governance and security for production AI

**Technical Capabilities:**

* RAG architecture for grounding LLMs in proprietary data
* Automated data science pipelines for knowledge updates
* OpenShift AI platform for building production AI applications
* Dual workflows for admins and data scientists

**Live Demonstration:**

* End-to-end RAG implementation on OpenShift AI
* Automated quality validation and monitoring
* GitOps continuous delivery with Argo CD
* Production troubleshooting and best practices

== Know/Show Methodology

Each demo module separates business context from technical demonstration:

**Know Sections** provide:

* Business challenges and value propositions
* Strategic insights for decision makers
* Why this technology matters to the organization

**Show Sections** provide:

* Step-by-step presenter instructions
* What to demonstrate and expected outcomes
* Talking points and objection handling

**Benefits:** Presenters understand business value before demonstrating features, enabling consistent messaging across different audiences and presentation styles.

== Demo Environment

**What's Deployed:**

* Red Hat OpenShift Container Platform (Kubernetes)
* Red Hat OpenShift AI 2.9 (complete AI/ML platform)
* PostgreSQL with pgvector (vector database for semantic search)
* LLM model serving (Granite, Mistral, or CodeLlama)
* Gradio chatbot interface (user-facing application)
* MinIO S3 storage (pipeline artifacts)
* Argo CD (GitOps continuous delivery)
* Tekton Pipelines (workflow automation)
